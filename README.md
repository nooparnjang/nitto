# แบบทดสอบความเครียด

## Overview
This project is an interactive web-based stress assessment presented as a story-driven experience.  
Users progress through scenes and make choices that affect their final stress score.

---

## Features
- Interactive story flow
- Multiple choice decisions
- Score calculation system
- Dynamic result based on user choices
- Responsive design (mobile-friendly)
- Background music support

---

## Tech Stack
- HTML5
- CSS3 (Flexbox / Auto Layout)
- JavaScript (ES Modules)
